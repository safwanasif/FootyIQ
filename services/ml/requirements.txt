# ============================================================================
# FootyIQ ML Service – Container Runtime Dependencies (requirements.txt)
# ============================================================================
# PURPOSE:
#   Minimal dependency set required to RUN the inference API (app.py) inside
#   the Docker container. This intentionally excludes ETL/training-only
#   libraries (statsbombpy, torch) to keep the production image lean —
#   the container only serves predictions from a pre-trained artifact,
#   it never trains or extracts data.
#
# NOTE FOR LOCAL DEV:
#   If you re-run etl.py or train_baseline.py locally against this trimmed
#   file, statsbombpy/torch will be missing. Reinstall them ad hoc for local
#   training work, or maintain a separate requirements-dev.txt going forward.
# ============================================================================
